🍔 Big Jay’s Eatery – Restaurant Ordering App

A mobile-style restaurant ordering app built with JavaScript, HTML, and CSS as part of my learning journey in the Fullstack Developer Path.

The app allows users to browse a menu, add items to their order, view the total cost, and complete payment through a checkout modal.

⸻

🚀 Features
	•	Dynamic menu rendering using JavaScript
	•	Add and remove items from the order
	•	Automatic meal deal discounts
	•	Real-time order total calculation
	•	Payment modal with required form validation
	•	Order confirmation message
	•	Modular project structure with reusable pricing logic

⸻

🧠 Technologies Used
	•	HTML5
	•	CSS3
	•	JavaScript (ES Modules)
	•	Vite (development server)

⸻

⸻

🛠 Getting Started
Install dependencies and start the development server:
npm install
npm run dev

Then open the local server shown in your terminal (usually):
http://localhost:5173
